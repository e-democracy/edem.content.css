Introduction
===========

This module customizes the egg provided CSS provided by `GroupServer`_. It,
however, does not provide the actual CSS used by forums.e-democracy.org.

Anybody interested in the CSS that is used by forums.e-democracy.org should
see `e-democracy-css`_.
